# CoolProp API endpoint

This is a simple page created using Flask 3 that allow the user to take full advantage of the CoolProp PropsSI functions through an API endpoint.


## Demo

https://coolprop-api.vercel.app/


## Running Locally

First, install the required libraries:

```bash
pip install -r requirements.txt
```

After requirements are fulfilled, just run the python file:

```bash
python api\index.py
```

Your Flask application is now available at `http://localhost:3000`.
